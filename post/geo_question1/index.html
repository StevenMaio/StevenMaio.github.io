A Geometry Question

Jul 17, 2020

In my work with a professor I've come across this problem and I can't solve it.

Problem: Suppose that $\gamma : I \to \mathbb{R}^2$ ($I$ is some interval) is a simple convex closed planar cuve, whose curvature is bounded above and below by positive constants, that is $$ 0 < A \le k(s) \le B < \infty $$ for $s \in I$, and let $D$ be the open region bounded by $\gamma$. Then there exists $c > 0$ such that for any $x \in D$ and $r < \text{diam } \gamma$ we have that $$ m_2 ( B(x;r) \cap D) \ge c r^2, $$ where $m_2$ is 2-d Lebesgue measure and $B(x;r) = \{ y \in \mathbb{R}^2 \ : \ | x - y | < r \}$.

I posted the question here. One of the important points is that because the curvature is bounded above, this limits how fast $\gamma$ can turn, which means that for those points inside $D$ really close to $\gamma$, there will be more in the opposite direction. I think this suggests that for some any point $x \in \gamma$, we can find some disk whose radius relates to $B$ contained inside $D$ and which intersects $\gamma$ only at $x$.
